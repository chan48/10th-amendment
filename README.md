# 10th-amendment
2018년 제10차 개헌의 문재인 대통령 개헌안 비교 페이지입니다.

## 생성 방법
직접 손으로 원본-개헌안 매핑 테이블을 작성하고 flask를 이용해 정적 html 페이지를 생성했습니다.

프론트엔드는 빠른 개발을 위해 semantic-ui를 이용했습니다.

생성에 사용된 flask 코드와 매핑 db 정보는 추후에 코드 정리 후 올라올 예정입니다.
